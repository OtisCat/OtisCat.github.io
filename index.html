Open Bender

When starting out on the Greens Farms Robotics team, I was tasked with creating a website. Used by the team for a number of years, the website has been a very important tool for managing journal entires. It is now open source on Github, meaning that other teams in the league can use it for themselves, and add features for everyone.

Billabung

Test your compatability with anyone by dragging a photo of yourself and that person into the designated slots. Finally, press the 'Do We Match?' button to see if you are compatible with them. This website was made in under 30 minutes during a static website hackathon.

Crash My Browser

"Crash My Browser" was made during the same competition as Billabung. It was done in an attempt to build a consistent User Interface style. The premise of the app is that the user can press a button, and if they do so, the web browser will crash.

GazLaxe

GazLaxe is a tool that gives everyday users the power of hackers. It was inspired by the Steve Jobs quote, "What if, instead of being in the right hands, it was in all the hands." In its current state, it allows users to perform SSH and FTP connections. It is open source, and no longer under active development due to the currently poor support for GTK.
